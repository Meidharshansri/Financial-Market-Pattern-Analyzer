# Financial Market Pattern Analyzer


A modular console-based analytics system that analyzes historical financial datasets (stocks, gold, silver) to detect market trends and volatility.

---

##  Features

*  Uptrend & Downtrend Detection
*  Volatility Analysis
*  Commodity Price Analysis
*  Multi-stock Dataset Support
*  OOP-based Modular Design

---

##  Technologies Used

* Python
* Object-Oriented Programming
* File-based Database
* Data Analytics Concepts
